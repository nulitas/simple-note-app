# Simple Note App

> A minimal note-taking application setup built with React, Vite, and HMR.

## ✨ Features
- Fast Refresh and Instant Server Start via Vite
- Component-based note management UI
- Clean and modern routing and state management (if applicable)

## 🚀 Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm run dev
   ```

## 🛠️ Built With
- **React.js**
- **Vite**
